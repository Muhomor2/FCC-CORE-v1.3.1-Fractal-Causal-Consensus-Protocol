# Contributing (restricted)

Because this project is distributed under a **No-Derivatives** license, you must NOT submit modified code as a forked derivative work.

Allowed ways to contribute:
1. Open an Issue with:
   - bug report
   - mathematical critique
   - suggested patch *as a description* (not as a modified repository)
2. Provide analysis, commentary, or independent replication results.

If the Author chooses, changes may be incorporated into the canonical repository by the Author only.

By submitting feedback, you confirm it is non-commercial and does not violate the ethical restrictions in `LICENSE.txt`.
